## 開發或正式環境


```
npm install

common options: [-S|--save|-D|--save-dev]
```


查看環境變數：

```sh
env
```

設定環境變數：

```sh
export NODE_ENV=development
# 或
export NODE_ENV=production
```

節點程式包管理器默認使用 `NODE_ENV=development`。
但 `process.env.NODE_ENV` 取得值仍是 `undefined`。
