(static) arguments(desc)

設定參數訊息。
以 `<>` 或 `[]` 為指令設定參數值描述，其結果將由 module:commander.action 接收。
注意！ 當設定此參數時，其後跟隨的參數視為拋棄。 
假設設定： `command  `
用戶輸入： `./program command a b c ...`
得到的結果： `a, <Command>, c, ...`
// `b` 不見了！ // 若未設定，則其所得的結果為： `a, c, ..., <Command>`

Name Type Description
desc String 參數訊息。
參數值以 `<自訂描述>` 表示必須，`[自訂描述]` 表示可選，如：`command <str>`。

(static) option(flag, description, handleopt , defaultValueopt)

設定選項（旗標）。
* 若 `handle` 和 `defaultValue` 皆未設定，其為布林值。
注意！ 當主程式有相同選項事件時，附屬指令的事件將被忽略。
Parameters:
Name Type Attributes Description
flag String 選項名。
* 以 `-n` 單槓一單字，`--name` 或 `--name-long` 雙槓完整命名。
* 以 `,` 或 `|` 做分割，簡短旗標在前，空格不限，如：`-n, --name`。
* 若其後有參數需加以告知，否則其後跟隨值將被略過。
* 其擁有值僅限一個（不可有空格符）。
* 參數值以 `<自訂描述>` 表示必須，`[自訂描述]` 表示可選，如：`-n, --name <str>`。
注意！ 雖說是可選，但其無值時卻無法觸動校驗函式。
description String 描述。
handle function | RegExp <optional>
對使用者輸入的值進行校驗。
defaultValue * <optional>
旗標的預設值。

Examples 
program
    .option( '-o, --optional [value]', '可選值。' )
    .option( '    --love <name>', '最喜歡 <name>。', 'You' )
    .option( '-i, --integer <n>', '正整數值。', parseInt )
    .option( '-c, --count', '累進數值。', increaseCount, 0 )
    // .option( '-c, --count [n]', '累進數值。', increaseCount, 0 )
    .option( '-r, --range <n>', '介於 2 到 5 的範圍。', /^(3|4)$/, '---' )
    .parse( process.argv )
;

function increaseCount( numChoA, numTotal ) {
    if ( numChoA ) numTotal += parseFloat( numChoA );
    else numTotal++;
    return numTotal;
}
program
    .option( '-f, --flag', '旗標事件。', flagBelong( '主旨令' ) )
    .command( 'testBA' )
        .option( '-f, --flag', '旗標事件。', flagBelong( 'testBA' ) )
    .parse( process.argv )
;

function flagBelong( strName ) {
    return function () {
        console.log( '旗標事件： ' + strName + ' --flag' );
    };
}
